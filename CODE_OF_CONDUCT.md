# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its Code of Conduct.

All participants in this project are expected to uphold a welcoming, inclusive, and respectful environment.

The full text of the Contributor Covenant v2.1 is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness unacceptable behavior, please open an issue or contact the maintainers directly.
